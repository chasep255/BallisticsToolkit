About Ballistics Toolkit

Project Overview

The Ballistics Toolkit is a suite of precision shooting tools — a ballistic calculator, target simulator, and interactive F‑Class simulator — designed to be accurate, fast, and easy to use. Everything runs in your browser with a high‑performance WebAssembly core: plan dope, practice wind, and simulate matches without installing software.

Features

Ballistic Calculator

Complete trajectory calculations with environmental factors, wind conditions, multiple drag models, and spin effects.

Target Simulator

Monte Carlo simulation with statistical variability modeling, and realistic accuracy assessment

Bullet Parameters

Complete bullet specification including weight, length, diameter, ballistic coefficient, and barrel twist rate for accurate calculations

WebAssembly

High-performance C++ algorithms compiled to WebAssembly

Wind Simulator

Real‑time wind field driven by multi‑octave curl noise (Simplex) with adjustable spatial/temporal scales, optional threshold gating, and global advection. Powers flags, mirage, and drift in the F-Class Simulator.

F-Class Simulator

Interactive 3D F‑Class match simulation (300–1000 yards). Real match flow (3× relays with sighters and record fire), mirage effect for wind reading, reactive wind flags, procedural terrain, and a 3D HUD with per‑relay stats and Scorecard.

How It Works

  • WebAssembly: Emscripten compilation for fast performance
  • Algorithms: 2nd-order Runge-Kutta (RK2) integration for trajectory calculations
  • Drag Models: G1 and G7 standard drag functions
  • Atmosphere: Uses temperature, humidity, and altitude. Pressure is derived from altitude unless provided by an API source.
  • Spin Aerodynamics: 4DOF trajectory with simplified, empirically tuned spin effects.
  • Wind Modeling: 2D curl field from Simplex noise with multi-octave components.

Why Ballistics Toolkit

Plan & Train: Build dope cards, visualize wind effects, and practice match flow from 300–1000 yards. Fast & Portable: Runs entirely in the browser — no login, no install.
